Help - Forum Rule Engine

Selecting the active user on the first pannel marks the user in whose name you will be performing all following actions. In this example, they are quite limited, but enough to explain the funcionalities of our Rule Engine.

These functionalities include writing a new post, liking, disliking and reporting other posts (and switching the active user). Be careful not to spam, as it can suspend the "spammer" user, preventing them from any actions.

Post and user labels will upadte as you play around with the options. The simulated console is there to help you understand which rules are triggered and when.

You can alwasy reset the system by restarting the server Java Application.
